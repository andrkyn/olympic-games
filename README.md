## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/andrkyn/olympic-games.git
cd olympic-games
```

```bash
npm install
```

## Steps.  For start server.js

To start the express server, run the following

```bash
npm start
```

Test CSV output to json in console
